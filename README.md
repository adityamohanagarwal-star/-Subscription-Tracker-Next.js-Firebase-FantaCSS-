# Subscription Tracker

A Subscription Tracker built with Next.js, Firebase, and FantaCSS that helps users manage and monitor their recurring subscriptions in one place. Track expenses, renewal dates, and stay organized with an intuitive dashboard.

### Features

* Add, edit, and delete subscriptions
* Track monthly and yearly subscription costs
* Monitor upcoming renewal dates
* Secure data storage with Firebase
* Responsive and modern UI with FantaCSS

### Tech Stack

* Next.js
* Firebase
* FantaCSS
* JavaScript
* HTML & CSS

A simple and efficient solution for managing subscriptions, controlling recurring expenses, and avoiding missed renewal dates.
